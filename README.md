# Meu Projeto de Website com Rolagem Animada

Este é um site que criei seguindo um tutorial para aprender a implementar uma biblioteca de rolagem animada. O objetivo do projeto foi explorar a criação de efeitos de rolagem que deixam a navegação mais fluida e visualmente atraente. A biblioteca utilizada permite que elementos na página ganhem vida à medida que o usuário rola, criando uma experiência de navegação mais dinâmica e interativa.

## Tecnologias Utilizadas

- ![HTML](https://img.shields.io/badge/-HTML5-E34F26?logo=html5&logoColor=white&style=flat) **HTML**: Para a estrutura básica do site.
- ![CSS](https://img.shields.io/badge/-CSS3-1572B6?logo=css3&logoColor=white&style=flat) **CSS**: Para o estilo do site.
- ![JavaScript](https://img.shields.io/badge/-JavaScript-F7DF1E?logo=javascript&logoColor=black&style=flat) **JavaScript**: Para manipular os elementos e adicionar as animações.

## Como Funciona

À medida que você rola a página, diferentes elementos vão aparecendo e se movendo de formas variadas. Isso é feito de maneira suave, o que melhora a experiência do usuário e deixa o site mais envolvente. Este projeto foi uma ótima maneira de entender como animações de rolagem podem ser aplicadas de forma prática.

## Como Usar

1. **Clone o repositório**: `git clone https://github.com/seu-usuario/seu-repositorio.git`
2. **Abra o arquivo index.html no seu navegador**: Basta abrir o arquivo diretamente para ver as animações em ação.

Sinta-se à vontade para explorar o código e adaptar conforme necessário!
